# REPORT

## 1. Task 1 觀察：哪個系三年變化最大？
在 112 到 114 三年資料中，變化最大的是食品科學系，分別是 52、24、29 人，最大差距為 28 人。這個波動明顯高於多數系所，代表該系在三年間招生表現不穩定。我推測可能與當年度招生名額調整、升學趨勢變化，或考生對科系就業想像改變有關。

## 2. Task 2 觀察：澎湖縣佔比與第二名縣市
六年總計招生 3216 人，其中澎湖縣 211 人，約佔 6.56%。若只看有明確縣市對應的資料，台中市總人數最高（502），第二名是高雄市（337）。若把無法對應的郵遞區號「其他」也算進去，其他會是最大宗，因此解讀時要先說明統計口徑。

## 3. 自由觀察
我注意到「其他」類別的人數非常高（980），顯示原始資料中的郵遞區號品質或格式一致性仍有改善空間。這會直接影響縣市熱力圖的可解讀性，讓部分真實來源被歸入同一桶。後續若能補齊對照表、清理空值與異常格式，圖表結果會更接近真實分布。
